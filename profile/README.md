## Public/Private Boundary

This organization may publish public-safe standards, templates, review gates, synthetic examples, public status ledgers, and approved Foundation documents.

This organization does not publish donor data, student data, volunteer private data, private Foundation operations, precise sensitive infrastructure locations, security-sensitive NEURONA details, sealed YOSO-YAi LLC IP, customer data, private training corpora, secrets, tokens, trading strategies, financial performance claims, or unreleased operational details.

## Non-Claims

A public repository here does not imply live schools, live NEURONAs, live deployments, released datasets, released models, live Hugging Face Spaces, active public reports, active public services, donor outcomes, or program results. Those claims require reviewed public evidence and a `released` status label.

## Review Discipline

Human Foundation review remains upstream of public claims, real reports, governance records, program status changes, and release decisions. Codex may help produce scaffolds, but Codex is not architectural authority.
