# Public And Private Boundaries

## Purpose

This document defines what this repository may publish and what must remain outside the public GitHub organization.

## Public Material Allowed Here

| Category | Public Posture |
| --- | --- |
| Repository standards | `published` when committed here and pushed to the public org. |
| Templates and checklists | `published` / `scaffolded`; usable for review and drafting only. |
| Synthetic examples | `published` / `scaffolded`; never evidence of real operations. |
| High-level program or governance descriptions | `published` / `scaffolded` when reviewed for public safety. |
| Status ledgers | `published`; labels must use `planned`, `scaffolded`, `published`, `released`, `paused`, `deprecated`, or `private/not-public`. |

## Material Excluded From This Repository

| Boundary | Material That Remains `private/not-public` |
| --- | --- |
| Donor privacy | Donor identities, donor records, payment records, donor communications, and donor-sensitive notes. |
| Student and school privacy | Student records, family records, individualized recommendations, school private operations, and private partner notes. |
| Volunteer privacy | Volunteer personal data, schedules, private contact information, screening records, and internal coordination notes. |
| Foundation operations | Private operating plans, incident notes, private board materials, legal drafts, finance details, and unreleased reports. |
| NEURONA security | Exact sensitive locations, security-sensitive mesh details, private telemetry, PCB-sensitive details, and operational deployment details. |
| YOSO-YAi LLC and YOSOR | Sealed IP, customer data, private training corpora, product internals, tokens, secrets, and company private operations. |
| Hugging Face releases | Model weights, datasets, Spaces, cards for unreleased artifacts, private corpora, and release claims without review. |

## Non-Claims

This repository does not claim live schools, live NEURONAs, live cleanup deployments, released datasets, released models, live Spaces, active public reports, donor results, financial performance, client outcomes, or active services unless a specific reviewed public artifact says so with `released` status.
