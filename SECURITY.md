# Security

## Public Security Boundary

Do not open public issues or pull requests containing secrets, tokens, credentials, private infrastructure details, exact sensitive NEURONA locations, private telemetry, donor records, student records, volunteer private data, customer data, private training corpora, or sealed YOSO-YAi LLC IP.

## Reporting

If you find a security or privacy issue, do not disclose details publicly in this repository. Use GitHub private vulnerability reporting if enabled for the repository, or contact the Foundation through a private reviewed channel listed on 218.network when available.

## Repository Scope

This repository is public documentation and scaffolding. It is not a production incident channel, emergency service, donor system, student system, model-development workspace, trading system, or private operations workspace.
