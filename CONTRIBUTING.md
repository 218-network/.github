# Contributing

## Public Contribution Rule

This repository is part of the 218 Network Foundation public GitHub organization surface. Contributions must preserve Foundation institutional voice, status labels, and public/private boundaries.

## Before Opening A Pull Request

- Use only public-safe material.
- Mark every repo, report, program, artifact, model, dataset, Space, school, NEURONA, deployment, or service with an approved status label.
- Keep Alexandra Caussade, K-ussade, the 218 Network Foundation, 218.network, YOSO-YAi LLC, and YOSOR distinct.
- Use synthetic examples unless a real document has explicit Foundation publication approval.
- Do not include secrets, tokens, donor data, student data, volunteer private data, customer data, private training corpora, sealed IP, or security-sensitive NEURONA details.

## Review Required

Foundation review is required before publishing or changing:

- governance policy text presented as final;
- real reports, financial summaries, or donation disclosures;
- program status changes;
- public claims about deployments, schools, NEURONAs, models, datasets, Spaces, reports, outcomes, or services;
- cross-links to K-ussade proof surfaces or YOSO-YAi LLC materials.
